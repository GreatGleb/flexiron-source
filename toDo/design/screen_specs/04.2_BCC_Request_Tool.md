# УЛ 04.2: Спецификация «Инструмент BCC-запросов» (BCC Request Tool)

*Источник требований: Алгоритм работы ERP Flexiron — Блок 1.1, 3.3, Sitemap 4.2*

## 1. Концепция (Concept)
Инструмент для автоматической рассылки запросов актуальных цен всем потенциальным поставщикам выбранной категории через скрытую копию (BCC).

---

## 2. Экран выбора и формирования (Selection)
*   **Выбор категорий**: Мультиселект категорий металла (напр. Листы, Перемычки). 
    *   *Действие*: Система мгновенно подсчитывает количество активных поставщиков, привязанных к этим категориям.
*   **Список получателей**: Список компаний с чекбоксами (по умолчанию выбраны все), которые получат письмо.
    *   *Важно*: Менеджер может вручную одного исключить перед отправкой.

---

## 3. Шаблон сообщения (Email Content)
*   **Тема письма**: Авто-генерация (напр. «Запрос цен на металл [Дата] — InBox LT»).
*   **Тело письма**: Редактируемое текстовое поле с авто-подстановкой товаров:
    *   {{items_list}} — список позиций из раздела "Дефицит" выбранной категории.
    *   {{company_signature}} — подпись текущего менеджера.
*   **Вложения**: Возможность прикрепить PDF-чертеж или спецификацию.

---

## 4. Логика отправки (BCC Logic)
*   **Механизм**: Письмо отправляется **одной транзакцией** со всеми Email-ами в поле BCC. 
    *   *Цель*: Поставщики не должны видеть друг друга и понимать, кому еще отправлен запрос.
*   **Фиксация**: После нажатия «Отправить» система создает запись в истории:
    *   *Кому оправлено*, *Какие позиции запрашивались*, *Дата*.

---

## 5. Обработка ответов (Response Handling)
*   **Фиксация прайса**: В таблице запросов появляется кнопка «Принять ответ».
*   **Занесение в историю**: Менеджер вручную вводит присланную цену за кг/м/шт, и она падает в **Историю цен поставщика** (Блок 3.3 ToDo) для графиков аналитики.

---

## 6. Технические детали
*   **Интеграция**: Авто-выбор почтового сервера из настроек 6.1.
*   **Аудит**: Запись в лог: «Рассылка по категории [X] отправлена [Кол-во] поставщикам».
