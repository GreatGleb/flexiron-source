# УЛ 01.0: Base Layout & Dashboard (Базовый макет и Дашборд)

## 1. Концепция и Назначение (Concept & Purpose)
Единый каркас (Layout) для всего внутреннего рабочего пространства Flexiron. Включает в себя глобальную навигацию (Sidebar), панель инструментов (Header) и центральную рабочую область, которая по умолчанию отображает сводный Дашборд (Dashboard). 

Дизайн сохраняет эстетику **Modern Glassmorphism** (блюр, белые обводки, акцентный синий `#1890FF`), но адаптирован для повседневной плотной работы с данными (чуть меньше прозрачности для читабельности таблиц).

---

## 2. Глобальная структура (Global Structure)

### 2.1. Боковое меню (Sidebar) - Левая часть
* **Ширина:** Фиксированная, ~260-280px.
* **Стиль:** Темная полупрозрачная панель с размытием заднего фона (backdrop-filter: blur).
* **Элементы:**
  1. **Логотип:** Вверху слева, Flexiron (icon + text).
  2. **Навигационные ссылки (Иконка + Текст):**
     * 📊 Дашборд (Обзор) — *Активный пункт*
     * 📦 Склад (Остатки, Приемка)
     * 👥 CRM и Продажи
     * 🏷️ Номенклатура (Товары и Услуги)
     * 🛒 Снабжение
     * 💰 Финансы
     * 📈 Аналитика
  3. **Подвал сайдбара:** Ссылка на "Настройки" ⚙️ и компактный блок текущего пользователя (Аватар + Имя + Роль).

### 2.2. Верхняя панель (Header) - Верхняя часть 
* **Высота:** ~70-80px.
* **Элементы (слева направо):**
  1. **Заголовок текущей страницы** (Например: *Дашборд руководителя*).
  2. **Глобальный поиск (Search Bar):** Поиск по всей базе (заказы, клиенты, партии). Стилизован под полупрозрачный input с иконкой лупы.
  3. **Кнопка "Быстрое действие" (+):** Создать новый заказ или приход.
  4. **Уведомления (Bell):** Иконка колокольчика с красным бейджем 🔴.
  5. **Языковой модуль:** RU / EN / LT (переиспользуем наш готовый `lang-switcher`).

## 3. Центральная область: Дашборд (Main Content: Dashboard)

Первый экран, который видит директор/менеджер после входа в систему.

### 3.1. Карточки KPI (Top Metrics)
Ряд из 3-4 стеклянных карточек (Glass cards) с ключевыми бизнес-показателями:
* **Металла на складе:** (например, 1,245 Тонн) — зеленая динамика +5%
* **Продажи за Сегодня:** (например, €14,500) — синий акцент
* **Активных заказов:** (например, 34 шт) — желтый маркер ожидания
* **Критический дефицит:** (например, 12 позиций) — красный маркер

### 3.2. Основные виджеты (Widgets)
* **График (Chart):** Динамика отгрузок за неделю (реализация через красивый CSS/SVG градиент или плейсхолдер графика).
* **Таблица "Последние действия" или "Срочные заказы":**
  * Колонки: ID заказа, Клиент, Статус (Бейджи: В работе, Сборка, Отгружено), Сумма.

## 4. Технические требования (Technical Logic)
* **CSS Grid / Flexbox:** Строгий каркас `display: grid; grid-template-columns: 260px 1fr;` (или flexbox) на уровне `body`.
* **Переполнение (Overflow):** Sidebar фиксирован (sticky/fixed), Main Content вертикально скроллится. Внутренние таблицы имеют свой горизонтальный скролл на мобильных.
* **i18n (Локализация):** Все элементы меню и заголовки должны быть обернуты в `data-i18n` для использования нашего движка (`engine.js`).
