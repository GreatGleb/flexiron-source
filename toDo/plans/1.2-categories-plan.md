## ТЗ (выжимка из источников)

### Что такое Категория
Категория — это шаблон структуры для товаров. Категории иерархические: дочерняя категория **наследует** все поля родительской и добавляет свои. Примеры: «Металл» → «Листы» → «Алюминиевые листы».

### Страница списка (CategoriesPage)
- Список всех категорий (дерево или плоский список с отступами по уровню)
- Каждая категория: название, родительская категория, количество полей, количество товаров
- Кнопка «Создать категорию» → модал создания
- Клик по строке → переход в карточку категории (`CategoryCardPage`)
- Поиск по названию
- Удаление категории (с подтверждением через модал)

### Страница карточки (CategoryCardPage)
**Секция «Основное»:**
- Название категории (обязательное)
- Родительская категория (CustomSelect из списка существующих, опционально)
- Описание (textarea, опционально)

**Секция «Наследованные поля»** (read-only, из родительской категории):
- Список полей родителя (или цепочки родителей) — только просмотр
- Если нет родителя — секция скрыта

**Секция «Собственные поля»** (редактируемая):
- Список полей этой категории с drag-and-drop для изменения порядка
- Каждое поле: название, тип, обязательное/нет, кнопки «Edit» / «Delete»
- Кнопка «Добавить поле» → модал добавления поля

**Модал добавления/редактирования поля:**
- Название поля (обязательное)
- Тип поля (CustomSelect): `text` / `number` / `boolean` / `enum` / `email` / `date` / `file`
- Обязательное? (чекбокс)
- Для типа `enum`: список вариантов (TagInput)

**Save bar** (clean-slate):
- **Всегда видна** на карточке категории
- Кнопка «Сохранить» — `disabled` по умолчанию, активна при `isDirty || fieldsChanged`
- Кнопки «Отмена» + «Сохранить»
- Save шлёт PATCH /api/categories/:id с дельтой + PUT /api/categories/:id/fields (весь массив полей целиком)

### API endpoints (писать в admin-api-contract.md)
```
GET    /api/categories              → PaginatedResponse<Category>
GET    /api/categories/:id          → Category (с fields[])
POST   /api/categories              → Category
PATCH  /api/categories/:id          → Category
DELETE /api/categories/:id          → void
PUT    /api/categories/:id/fields   → CategoryField[] (bulk replace, для reorder и save)
```

### Типы
```ts
Category {
  id: string
  name: string
  parentId: string | null
  description: string | null
  fieldCount: number       // собственные поля
  productCount: number     // сколько товаров в этой категории
  inheritedFields: CategoryField[]   // поля родителей (read-only)
  fields: CategoryField[]            // собственные поля
}

CategoryField {
  id: string
  name: string
  type: 'text' | 'number' | 'boolean' | 'enum' | 'email' | 'date' | 'file'
  required: boolean
  order: number
  options: string[]   // только для type === 'enum', иначе []
}

CategoryFilters {
  search: string
}
```

### Промпт 7 — Composable: useCategoryCard (карточка)

```
Задача: создай src/composables/useCategoryCard.ts.

Composable для страницы карточки (clean-slate save):

export function useCategoryCard(id: string) {
  category: ref<Category | null>
  loading: ref<boolean>
  saving: ref<boolean>
  error: ref<string | null>
  { isDirty, track, diff, reset } из useDirtyCheck<Pick<Category,'name'|'parentId'|'description'>>()
  
  — localFields: ref<CategoryField[]>  — локальная копия fields для редактирования
  — fieldsChanged: computed<boolean>   — сравнение localFields с category.value.fields по JSON
  — isAnythingDirty: computed<boolean> — isDirty || fieldsChanged
  
  load(): async
    — category.value = await getCategory(id)
    — track({ name, parentId, description }) из category.value
    — localFields.value = structuredClone(category.value.fields)  // ВАЖНО: structuredClone, не reactive ref напрямую
  
  save(): async
    — если isAnythingDirty === false: return
    — saving.value = true
    — параллельно (Promise.all):
        1. если isDirty: patchCategory(id, diff({ name, parentId, description }))
        2. если fieldsChanged: putCategoryFields(id, localFields.value)
    — при успехе: reload (load()), reset dirty, toast «Сохранено»
    — при ошибке: toast с текстом ошибки
    — finally: saving.value = false
  
  discard(): — reload из сервера (вызов load())
  
  addField(field: Omit<CategoryField, 'id' | 'order'>): void
    — генерирует временный id ('tmp-' + Date.now()), order = localFields.length
    — push в localFields.value
  
  updateField(id: string, delta: Partial<CategoryField>): void
    — находит поле в localFields по id, merge delta
  
  deleteField(id: string): void
    — фильтрует localFields, пересчитывает order
  
  reorderFields(newOrder: CategoryField[]): void
    — заменяет localFields.value, пересчитывает order по индексу массива
  
  return { category, loading, saving, error, localFields, isAnythingDirty, isDirty, fieldsChanged,
           load, save, discard, addField, updateField, deleteField, reorderFields }
}

ВАЖНО: structuredClone на reactive данных вызывает DataCloneError.
Для клонирования reactive значений используй JSON.parse(JSON.stringify(...)).
```
