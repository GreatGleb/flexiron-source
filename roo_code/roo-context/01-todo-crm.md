# План разработки CRM-системы складского учета

> Источник: [`InBox LT CRM ToDo.md`](../toDo/InBox%20LT%20CRM%20ToDo.md)

**Вводная:** Детализированный план функций CRM и складской системы на базе анализа рабочих Excel-таблиц (учет металла, генерация обрезков, создание таможенных и локальных счетов).

---

## 1. СКЛАД

### 1.1 `page Товары` (Справочник номенклатуры)
- [ ] **Динамические категории:** Создание/редактирование/удаление категорий (Трубы, Листы, Декор. профили). Конструктор свойств — любое свойство (число, строка, список) привязывается к категории.
- [ ] **Сводный список (Таблица-каталог):** Общий список всех товаров из всех категорий (базовые колонки: Название, Базовая цена, Единицы измерения).
- [ ] **Карточка "вида товара":** Базовые данные + динамические поля категории. Эталонная стоимость (EUR/vnt, EUR/kg, EUR/m). **Привязка Поставщиков** внутри карточки товара.

### 1.2 `page Склад` (Физическое наличие, партии и обрезки)
- [ ] **Учет движения и партий:** Приход (закупка) с привязкой к эталонному "Товару". Расход (продажа/списание).
- [ ] **Динамические складские свойства:** Поля, не относящиеся к карточке товара (Состояние, Ячейка/Полка, Резерв).
- [ ] **Настраиваемое отображение таблиц (Views):** Общий список + просмотр по категории со специфическими колонками.
- [ ] **Система "Обрезков" (Atraižos):** Автоматический механизм отрезания части от целого. Учет технологических потерь (ширина реза). Калькуляция параметров обрезка. Фотографии/чертежи для нестандартных форм. Маркировка и QR-коды. Интеграция с лазерной резкой (OptiCut).

### 1.3 `page Услуги` (Справочник нематериальных позиций)
- [ ] Карточка услуги (Название, Тип). Тарификация (себестоимость + наценка).

## 2. CRM

### 2.1 `page Клиенты` (Справочник контрагентов)
- [ ] Базовая информация (реквизиты). Динамические свойства клиента. История работы (заказы, счета). Аналитика отказов.

### 2.2 `page Заказы` (Формирование отгрузок и генерация документов)
- [ ] Создание заказа (шапка): выбор клиента, тип комплекта (локальный/экспорт).
- [ ] Корзина заказа и связь со Складом: списание остатков, механизм "Отрезать", резервирование.
- [ ] Добавление Услуг в заказ. Гибкое ценообразование. Показ маржинальности.
- [ ] Финансовая и логистическая автоматизация: калькулятор габаритов, скидки, НДС.
- [ ] **Генератор печатных документов:** Proforma-Invoice, CMR, KPO, Сертификаты, Lipdukai. Ручная корректировка "на лету".
- [ ] Динамические свойства заказа. Кастомная воронка статусов и логика резерва. Управление временем резерва.

## 3. ЗАКУПКИ И ПОСТАВЩИКИ

### 3.1 `page Поставщики`
- [ ] Профили поставщиков. **Алгоритм определения дефицита (Supply Health):** BCC Request History, Deficit Items, Pricing History, Status поставщика. Порог 50+ баллов = `hasDeficit`. Визуализация рисков.

### 3.2 `page Потребности` (Low Stock Alerts)
- [ ] Лимиты неснижаемого остатка. Автоматический список закупок.

### 3.3 `Массовые запросы цен` (BCC Messaging)
- [ ] Скрытая рассылка в 1 клик. История цен.

## 4. СИСТЕМНАЯ АРХИТЕКТУРА И НАСТРОЙКИ

### 4.1 Пользовательские роли и права
- [ ] Матрица доступов (Администратор, Продажи, Склад, Бухгалтерия).

### 4.2 Облачная инфраструктура и Электронный архив
- [ ] Облачный VPS. Электронный архив PDF. Масштабируемость и API.

### 4.3 Мультиязычность
- [ ] Локализация (RU, LT, EN).

### 4.4 Журнал аудита (Audit Trail)
- [ ] Автоматическая фиксация всех действий. История в карточке. Защита от удаления.

## 5. СИСТЕМА УВЕДОМЛЕНИЙ
- [ ] Внутренние уведомления (колокольчик со счётчиком, прямые ссылки). Push-уведомления (Web Push / PWA). Настройка подписки.

## 6. КОНСТРУКТОР СВОЙСТВ (Admin Configuration)

### 6.1 Структура полей
- [ ] Разделы (Groups). Drag-and-drop порядок.

### 6.2 Типы свойств (Field Types)
- **Простые:** Текст, Число, Логическое, Email, Timestamp, Sequence, Color Picker
- **Идентификация:** Штрих-код/QR, Серийный номер, Геоданные
- **Временные:** Date Range, Temporal Data
- **Финансовые:** Money/Currency, Quantity with UOM
- **Структурные:** Enum, Hierarchy, Grid/Nested Table, BOM, Formula/Calculated Field
- **Управление:** ACL, State Machine
- **Медиа:** Фото, Аудио/Видео, Документ (PDF/Office)
- **Бизнес-объекты:** Reference, External Link/URL, Struct, Array of Structs

### 6.3 Фильтрация и сортировка
- [ ] Базовая (по общим полям). Расширенная (по спец. свойствам).

### 6.4 Глобальные справочники и константы
- [ ] Единые справочники (Валюты, Языки, Страны, UOM). Глобальные константы (НДС, маржа). Data Connectors (курсы валют). Визуализация списков (иконки, цвета). Правила пересчета (UOM Conversion). Редактор бизнес-логики.

### 6.5 Принцип максимально простого UX
- [ ] Zero-Code Admin Panel. Простота для конечного пользователя. "Не повторить ошибки Excel".
