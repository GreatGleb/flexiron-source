# УЛ 05.1: Header (Верхняя панель)

## 1. Концепция и Назначение (Concept & Purpose)
Верхняя панель системы Flexiron. Отвечает за глобальный поиск, уведомления, переключение локализации и отображение профиля пользователя.

**Ключевой принцип дизайна:** Сохраняет единое визуальное пространство с лендингом. **За панелью виден размытый фон** — промышленная фотография (лазерная резка металла, яркие искры, синие и оранжевые тона). Сама панель — **светлое матовое стекло** (`rgba(255, 255, 255, 0.12)` + `backdrop-filter: blur(25px)`) с тонкой нижней белой обводкой.

## 2. Визуальный стиль (Design Tokens)
* **Фон панели:** Светлый / молочный Glassmorphism: `background: rgba(255, 255, 255, 0.12)`, `border-bottom: 1px solid rgba(255, 255, 255, 0.25)`.
* **Размытие:** `backdrop-filter: blur(25px)` — за панелью виден полупрозрачный фон страницы (фото).
* **Текст:** Белый (`#fff`) для заголовков, `rgba(255,255,255,0.7)` для второстепенного текста.
* **Акцент:** `#1890FF` (primary-blue) для фокуса инпута, активных состояний и иконок.

## 3. Структура и Элементы
* **Высота:** ~75px. Полная ширина (правее сайдбара).
* **Элементы (слева → направо):**
  1. **Заголовок страницы** — белый, `font-weight: 600`, 1.25rem. Динамически подставляется (напр. "Дашборд").
  2. **Глобальный поиск (Search Bar):**
     - Полупрозрачный input (`rgba(255,255,255,0.1)`, border `rgba(255,255,255,0.2)`).
     - Иконка лупы слева. Плейсхолдер белый/прозрачный.
     - При фокусе: border `#1890FF`, лёгкое расширение.
  3. **Переключатель языков:** RU / EN / LT — кнопки в том же стиле, что на лендинге.
