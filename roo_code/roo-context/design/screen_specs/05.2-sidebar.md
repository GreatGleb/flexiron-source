# УЛ 05.2: Sidebar (Боковое меню)

## 1. Концепция и Назначение (Concept & Purpose)
Главная навигационная панель системы Flexiron. Зафиксирована слева. Обеспечивает доступ ко всем основным модулям системы.

**Ключевой принцип дизайна:** Единое визуальное пространство с лендингом — **за сайдбаром виден размытый индустриальный фон** (фото с производства). Сама панель выполнена как **вертикальная светлая стеклянная плита** (`rgba(255, 255, 255, 0.1)` + сильный `backdrop-filter: blur`), с тонкой правой белой обводкой.

## 2. Визуальный стиль (Design Tokens)
* **Фон панели:** `background: rgba(255, 255, 255, 0.1)` — точно как на лендинге.
* **Обводка (inner glow):** `border-right: 1px solid rgba(255, 255, 255, 0.35)` + `box-shadow: inset -1px 0 0 rgba(255,255,255,0.2)`.
* **Размытие:** `backdrop-filter: blur(20px)` — за панелью просвечивает размытое фото.
* **Текст:** Белый (`#fff`) для активного пункта и логотипа. `rgba(255,255,255,0.65)` для неактивных.
* **Акцент:** `#1890FF` — левая полоса активного пункта, фон активного пункта `rgba(24,144,255,0.15)`.

## 3. Структура и Элементы
* **Ширина:** ~260–280px. Высота: 100vh.
* **Разделы Сайдбара:**
  1. **Шапка:** Логотип Flexiron (SVG белый). Граница снизу `rgba(255,255,255,0.1)`.
  2. **Навигационные ссылки** (SVG-иконка + текст, в порядке Sitemap):
     - 📊 Дашборд *(активный — синий фон + левая accentная полоска)*
     - 🏷️ Товары (Раздел 1 Sitemap — Номенклатура)
     - 📦 Склад (Раздел 2)
     - 👥 Продажи и CRM (Раздел 3)
     - 🛒 Поставки (Раздел 4 — поставщики, BCC)
     - 💰 Финансы (Раздел 5)
     - 📈 Аналитика (Раздел 7 — KPI, отчёты)
     - Hover: `rgba(255,255,255,0.08)`.
  3. **Подвал Сайдбара:**
     - Ссылка "Настройки" (иконка шестерёнки).
     - Граница сверху `rgba(255,255,255,0.1)`.
